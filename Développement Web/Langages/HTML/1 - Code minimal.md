# Exemple de code minimal

```html
<!DOCTYPE html>
<html lang="fr">
<head>
	<!-- encodage à indiquer au début du head -->
	<meta charset="utf-8" />
	<!-- réglage du viewport sur les appareils mobiles -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<!--le titre est obligatoire pour un code valide -->
	<title>Titre de la page </title>
	<!-- feuille de style -->
	<link rel="stylesheet" href="style.css" />
</head>
<body>
	<!-- script JS -->
	<script src="script.js"></script>
</body>
</html>
```
