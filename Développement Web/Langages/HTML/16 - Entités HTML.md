# Entités html

Caractères permettant de remplacer les caractères réservés de HTML ainsi que les caractères invisibles

- Quelques entités :

	
	| Caractère        | Entité   |
	| ---------------- | -------- |
	| Espace insécable | `&nbsp;` |
	| `<`              | `&lt;`   |
	| `>`              | `&gt;`   |
	| `&`              | `&amp;`  |
	| `"`              | `&qot;`  |
	| `'`              | `&apos;` |
	| `€`              | `&euro;` |
