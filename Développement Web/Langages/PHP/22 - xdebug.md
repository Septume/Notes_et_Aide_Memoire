# Xdebug

## **Installation**

Procédure d'installation :

**[https://xdebug.org/wizard.php](https://xdebug.org/wizard.php)**

## **Configuration**

Fichier `php.ini`

```
 display_errors = On
 html_errors = On
 error_reporting = E_ALL
 
 [xdebug]
 ;autoriser le débogage avec un client distant
 xdebug.remote_enable = 1
```
